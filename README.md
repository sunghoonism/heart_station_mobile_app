# 마음 충전소

당신의 이야기에 엉뚱하지만 진심 담은 응원을 보내는 AI 기반 모바일 애플리케이션입니다.

## 주요 기능

- **AI 기반 응원 메시지 생성**: Groq의 Gemma-2 모델을 활용하여 사용자의 이야기에 맞는 따뜻한 응원 메시지를 생성합니다.
- **응원 메시지 저장**: 생성된 응원 메시지를 저장하고 나중에 다시 볼 수 있습니다.
- **응원 메시지 공유**: 받은 응원 메시지를 다른 사람들과 공유할 수 있습니다.
- **앱 공유**: 마음 충전소 앱을 다른 사람들과 공유할 수 있습니다.

## 기술 스택

- **프레임워크**: Flutter
- **언어**: Dart
- **캐릭터 디자인**: Google AI Studio
- **AI 모델**: Groq Gemma-2

## 설치 방법

1. Flutter SDK 설치
2. 프로젝트 클론
```bash
git clone https://github.com/sunghoonism/heart_station_mobile_app.git
```
3. 의존성 설치
```bash
flutter pub get
```
4. 앱 실행
```bash
flutter run
```

## 환경 설정

- Flutter SDK: 3.0.0 이상
- Dart SDK: 2.17.0 이상
- Android Studio / VS Code
- Android SDK / Xcode (iOS 개발 시)
